# Vehicle System

## Descrição

O Vehicle System é um sistema simples desenvolvido em Java com o objetivo de aplicar conceitos fundamentais de Programação Orientada a Objetos (POO).

O projeto simula o gerenciamento básico de veículos, permitindo a criação e manipulação de diferentes tipos, como carros e motos.

 ## Objetivos do Projeto

Este projeto foi desenvolvido com foco em aprendizado, abordando:

- Herança
- Polimorfismo
- Encapsulamento
- Uso de construtores
- Organização em pacotes
- Interação com o usuário via terminal

## Estrutura do Projeto
```
vehicle-system/
│
├── src/
│   ├── app/
│   │   └── Main.java
│   │
│   ├── model/
│   │   ├── Veiculo.java
│   │   ├── Carro.java
│   │   └── Moto.java

```

## Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)

## Funcionalidades

- Criar veículos (Carro e Moto)
- Definir atributos como:
  - Placa
	-	Tipo
	-	Outras características específicas
- Exibir informações dos veículos

## Como Executar o Projeto

1.	Clone o repositório:<br>
`git clone https://github.com/lucascarvalho-oliveira/vehicle-system.git`
2. Acesse a pasta do projeto:<br>
`cd vehicle-system`
3. Compile o projeto:<br>
`javac src/app/Main.java`
4. Execute<br>
`java src.app.Main`

## Conceitos Aplicados

### Herança

A classe Veiculo é utilizada como base para outras classes, como Carro e Moto.

### Polimorfismo

Permite tratar diferentes tipos de veículos de forma genérica.

### Encapsulamento

Os atributos são protegidos e acessados por métodos getters e setters.
